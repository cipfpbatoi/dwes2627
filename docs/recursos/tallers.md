# Tallers

Els tallers són sessions curtes de suport per desbloquejar decisions tècniques concretes. No són microreptes nous ni tenen nota separada: ajuden a millorar el treball que ja estàs fent dins d'un repte.

## Tallers fixos

| Taller | Quan et pot ajudar |
|---|---|
| [MT01. Projecte no CRUD](Tallers/MT01-Projecte-no-CRUD.pdf) | Quan el projecte sona massa genèric o només planteja altes, baixes i llistats. |
| [MT02. README executable i commit defensable](Tallers/MT02-README-executable-i-commit-defensable.pdf) | Quan cal fer que una altra persona puga arrancar i revisar el projecte. |
| [MT03. Validació de servidor amb casos roïns](Tallers/MT03-Validacio-de-servidor-amb-casos-roins.pdf) | Quan el formulari només funciona en el cas correcte. |
| [MT04. Sessió, cookies i estat](Tallers/MT04-Sessio-cookies-i-estat.pdf) | Quan el flux necessita recordar informació entre peticions. |
| [MT05. Auth mínima defensable](Tallers/MT05-Auth-minima-defensable.pdf) | Quan cal protegir una operació real sense muntar un sistema d'usuaris massa gran. |
| [MT06. Debugging i prova curta](Tallers/MT06-Debugging-i-prova-curta.pdf) | Quan cal provar i explicar un flux abans de continuar afegint capes. |
| [MT07. Migrar un cas d'ús a framework](Tallers/MT07-Migrar-un-cas-dus-a-framework.pdf) | Quan toca portar un flux de `R2` a framework en `R3`. |
| [MT08. Migracions, seeders i dades de prova](Tallers/MT08-Migracions-seeders-i-dades-de-prova.pdf) | Quan cal reconstruir la BBDD i carregar dades inicials de manera reproduïble. |
| [MT09. On pose la lògica](Tallers/MT09-On-pose-la-logica.pdf) | Quan no està clar què va en ruta, controlador, servei, model o vista. |
| [MT10. Dissenyar endpoint abans de programar](Tallers/MT10-Dissenyar-endpoint-abans-de-programar.pdf) | Quan cal definir contracte d'API abans d'escriure codi. |
| [MT11. Documentar i comprovar una API](Tallers/MT11-Documentar-i-comprovar-una-API.pdf) | Quan cal demostrar que productor i consumidor parlen el mateix contracte. |
| [MT12. Triar una integració amb valor](Tallers/MT12-Triar-una-integracio-amb-valor.pdf) | Quan cal seleccionar una font externa que aporte valor real al producte. |
| [MT13. Secrets, env i claus](Tallers/MT13-Secrets-env-i-claus.pdf) | Quan la integració necessita claus, tokens o configuració sensible. |
| [MT14. Mapping de resposta externa](Tallers/MT14-Mapping-de-resposta-externa.pdf) | Quan cal transformar una resposta externa perquè encaixe amb el domini propi. |
| [MT15. Defensa tècnica curta](Tallers/MT15-Defensa-tecnica-curta.pdf) | Quan cal preparar una explicació breu, concreta i verificable del treball. |
