# Rúbrica R4M4. Consum real del servei

| Dimensió | Pes | Què es comprova |
|---|---:|---|
| Resolució funcional | `30%` | Consumidor mínim, col·lecció o script que fa peticions reals. |
| Verificació | `25%` | Cas correcte i cas d'error o invàlid. |
| Traçabilitat | `10%` | Evidències del consumidor i de les proves. |
| Documentació | `15%` | Instruccions per reproduir el consum. |
| Ús d'IA | `5%` | Si hi ha IA, consumidor adaptat i verificat. |
| Qualitat tècnica | `15%` | El consumidor llig resposta, envia credencial si cal i tracta error mínim. |

## Regles dures

- Sense consumidor o prova equivalent, màxim recomanat `5`.
- Sense cas d'error, màxim recomanat `7`.
- Sense instruccions de reproducció, màxim recomanat `8`.
