# R2M8. Primera peça testable amb POO i Composer

## RA avaluat

- **RA avaluat**: `RA3`
- **CA avaluats**: `RA3.d`, `RA3.g`
- **RA de context**: `RA4.f`

Este microrepte genera nota només per al RA avaluat. Els RA de context poden aparéixer en el producte, però no es ponderen en esta rúbrica.

## Què es valora

Es valora que una regla, càlcul o comprovació del flux de `R2` passe a una classe simple carregada amb Composer o mecanisme equivalent, i que tinga una prova unitària mínima.

No es demana encara MVC complet, ORM, framework ni POO extensa.

## Evidències mínimes

- Flux de `R2M7` provat com a punt de partida.
- `composer.json` amb autoload o alternativa equivalent justificada.
- Ús de `vendor/autoload.php` o mecanisme equivalent.
- Classe simple en `src/` o carpeta equivalent.
- Classe vinculada a una regla, comprovació o càlcul real del projecte.
- La classe no depén directament de `$_POST`, `$_SESSION`, `$_COOKIE` ni HTML.
- Prova unitària mínima amb PHPUnit, Pest o script automàtic equivalent.
- Execució documentada de la prova.
- Prova de no regressió d'un cas clau del flux web.
- Nota de què queda per reorganitzar o ampliar en `R3`.

## Rúbrica

| Dimensió | Pes | Què ha d'estar clar |
|---|---:|---|
| Resolució funcional | `30%` | Hi ha autoload, classe real i el flux web continua funcionant. |
| Verificació | `20%` | Hi ha prova unitària mínima i prova de no regressió. |
| Traçabilitat | `10%` | Hi ha registre del pas cap a Composer, classe i test. |
| Documentació | `15%` | S'explica com executar la prova, què fa la classe i què queda per a `R3`. |
| Ús d'IA | `5%` | Si s'ha usat IA, queda registrada la proposta acceptada i la verificació pròpia. |
| Qualitat tècnica | `20%` | La classe encapsula una regla real i no depén de superglobals ni HTML. |

## Regles dures

- Sense `composer.json`/autoload o equivalent justificat, màxim recomanat `7`.
- Sense classe pròpia real, màxim recomanat `6`.
- Sense prova unitària o script automàtic equivalent, màxim recomanat `7`.
- Sense prova de no regressió del flux web, màxim recomanat `8`.
- Si la classe només conté HTML, superglobals o codi buit, màxim recomanat `7`.
