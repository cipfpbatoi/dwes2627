# Guies d'avaluació dels microreptes

Estes guies indiquen com es revisarà cada microrepte. No substituïxen l'enunciat del repte: t'ajuden a comprovar si tens les evidències necessàries abans d'entregar.

Cada guia està alineada amb l'autocorrecció corresponent i mostra:

- visor del microrepte;
- criteris i pesos;
- evidències requerides;
- senyals esperats;
- elements fora d'abast;
- connexió amb l'autocorrecció.
