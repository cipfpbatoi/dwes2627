# R3M2. Persistència mínima amb migracions, fixtures/seeders o equivalent

## RA avaluat

- **RA avaluat**: `RA6`
- **CA avaluats**: `RA6.a`, `RA6.b`, `RA6.c`, `RA6.d`, `RA6.e`
- **RA de context**: `RA5.f`

Este microrepte genera nota només per al RA avaluat. Els RA de context poden aparéixer en el producte, però no es ponderen en esta rúbrica.


## Visor del microrepte

| Camp | Valor |
|---|---|
| Repte | `R3` |
| Sessió | `R3S2` |
| Microrepte | `R3M2` |
| Pes dins del repte | `18%` |
| Autocorrecció | `r3-s02-persistencia-minima-migrations-seeders` |

## Criteris i pesos

| Dimensió | Pes | Què es comprovarà |
|---|---:|---|
| Resolució funcional | `30%` | BBDD real amb migracions o equivalent, dades reproduïbles i lectura de dades. |
| Verificació | `20%` | Reset o reconstrucció de BBDD, càrrega i lectura comprovades. |
| Traçabilitat | `10%` | Commits o registre de model, migracions/càrrega de dades i prova. |
| Documentació | `15%` | Com crear BBDD, carregar dades i quin flux alimenten. |
| Ús d'IA | `5%` | Registre i validació de propostes de camps, migracions o càrrega inicial. |
| Qualitat tècnica | `20%` | Esquema mínim, coherent i no manual. |

## Evidències requerides

- Connexió a BBDD.
- Migracions o equivalent.
- Fixtures/seeders o script equivalent.
- Dades inicials reproduïbles.
- Lectura real de conjunt de dades.
- Ús o preparació del conjunt en un flux.

## Fora d'abast

No es demanen encara dos fluxos complets ni API.
