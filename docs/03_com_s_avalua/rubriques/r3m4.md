# R3M4. Segon cas d'ús o ampliació funcional

## RA avaluat

- **RA avaluat**: `RA8`
- **CA avaluats**: `RA8.f`, `RA8.g`
- **RA de context**: `RA5.e`, `RA5.f`, `RA5.g`, `RA5.h`, `RA6.c`, `RA6.d`, `RA6.f`

Este microrepte genera nota només per al RA avaluat. Els RA de context poden aparéixer en el producte, però no es ponderen en esta rúbrica.


## Visor del microrepte

| Camp | Valor |
|---|---|
| Repte | `R3` |
| Sessió | `R3S4` |
| Microrepte | `R3M4` |
| Pes dins del repte | `12%` |
| Autocorrecció | `r3-s04-segon-cas-us-ampliacio-funcional` |

## Criteris i pesos

| Dimensió | Pes | Què es comprovarà |
|---|---:|---|
| Resolució funcional | `30%` | Segon flux o ampliació funcional amb BBDD i canvi dinàmic. |
| Verificació | `20%` | Prova del segon flux, error i regressió del primer. |
| Traçabilitat | `10%` | Commits o registre del segon flux i regressió. |
| Documentació | `15%` | Què aporta i com provar-lo. |
| Ús d'IA | `5%` | Registre d'idees o codi generat si aplica. |
| Qualitat tècnica | `20%` | Reutilitza estructura sense duplicació buida. |

## Evidències requerides

- Segon flux descrit.
- Ruta i controlador.
- BBDD real.
- Resposta dinàmica.
- Validació o error.
- Regressió del primer flux.

## Fora d'abast

No es demanen API, integració externa ni rols complexos.
