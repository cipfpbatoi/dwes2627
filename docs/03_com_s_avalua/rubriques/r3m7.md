# R3M7. Tancament tècnic

## RA avaluat

- **RA avaluat**: `RA5`
- **CA avaluats**: `RA5.f`, `RA5.g`
- **RA de context**: `RA6.g`

Este microrepte genera nota només per al RA avaluat. Els RA de context poden aparéixer en el producte, però no es ponderen en esta rúbrica.


## Visor del microrepte

| Camp | Valor |
|---|---|
| Repte | `R3` |
| Sessió | `R3S7` |
| Microrepte | `R3M7` |
| Pes dins del repte | `13%` |
| Autocorrecció | `r3-s07-tancament-tecnic` |

## Criteris i pesos

| Dimensió | Pes | Què es comprovarà |
|---|---:|---|
| Resolució funcional | `30%` | Projecte arrancable, BBDD reproduïble i dos fluxos demostrables. |
| Verificació | `20%` | Instruccions provables d'arrencada, reconstrucció de BBDD, càrrega inicial i fluxos. |
| Traçabilitat | `10%` | Commits finals, decisions i backlog. |
| Documentació | `15%` | README amb arrencada, BBDD, fluxos, protecció d'accessos, demo i pas a `R4`. |
| Ús d'IA | `5%` | Verificació del text generat si s'ha usat IA. |
| Qualitat tècnica | `20%` | Tancament honest: migrat, ampliat, pendent i base defensable. |

## Evidències requerides

- README tècnic.
- Instruccions d'arrencada.
- Instruccions de BBDD.
- Instruccions per provar l'acció protegida.
- Dos fluxos descrits.
- Decisions tècniques.
- Backlog cap a `R4`.

## Fora d'abast

No s'ha d'implementar encara l'API de `R4`.
