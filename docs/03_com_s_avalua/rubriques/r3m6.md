# R3M6. Tancament tècnic

## Visor del microrepte

| Camp | Valor |
|---|---|
| Repte | `R3` |
| Sessió | `R3S6` |
| Microrepte | `R3M6` |
| Pes dins del repte | `15%` |
| Autocorrecció | `r3-s06-tancament-tecnic` |

## Criteris i pesos

| Dimensió | Pes | Què es comprovarà |
|---|---:|---|
| Resolució funcional | `30%` | Projecte arrancable, BBDD reproduïble i dos fluxos demostrables. |
| Verificació | `20%` | Instruccions provables d'arrencada, migrations, seeders i fluxos. |
| Traçabilitat | `10%` | Commits finals, decisions i backlog. |
| Documentació | `15%` | README amb arrencada, BBDD, fluxos, demo i pas a `R4`. |
| Ús d'IA | `5%` | Verificació del text generat si s'ha usat IA. |
| Qualitat tècnica | `20%` | Tancament honest: migrat, ampliat, pendent i base defensable. |

## Evidències requerides

- README tècnic.
- Instruccions d'arrencada.
- Instruccions de BBDD.
- Dos fluxos descrits.
- Decisions tècniques.
- Backlog cap a `R4`.

## Fora d'abast

No s'ha d'implementar encara l'API de `R4`.
