# R3M6. Qualitat i estabilització

## RA avaluat

- **RA avaluat**: `RA5`
- **CA avaluats**: `RA5.g`, `RA5.h`
- **RA de context**: `RA6.f`, `RA6.g`

Este microrepte genera nota només per al RA avaluat. Els RA de context poden aparéixer en el producte, però no es ponderen en esta rúbrica.


## Visor del microrepte

| Camp | Valor |
|---|---|
| Repte | `R3` |
| Sessió | `R3S6` |
| Microrepte | `R3M6` |
| Pes dins del repte | `12%` |
| Autocorrecció | `r3-s06-qualitat-estabilitzacio` |

## Criteris i pesos

| Dimensió | Pes | Què es comprovarà |
|---|---:|---|
| Resolució funcional | `30%` | Fluxos principals funcionen amb BBDD, errors controlats i protecció intacta. |
| Verificació | `20%` | Checklist o proves amb cas positiu, negatiu, reconstrucció de BBDD, càrrega inicial i acció protegida. |
| Traçabilitat | `10%` | Incidències, correccions i commits. |
| Documentació | `15%` | Proves, errors i estat dels fluxos. |
| Ús d'IA | `5%` | Registre si s'ha usat per depurar o proposar proves. |
| Qualitat tècnica | `20%` | Revisió de responsabilitats i estabilització real. |

## Evidències requerides

- Checklist o proves dels fluxos principals.
- Cas positiu i negatiu.
- Error controlat.
- Migracions o equivalent i càrrega inicial comprovades.
- Protecció d'accessos comprovada.
- Incidències corregides.

## Fora d'abast

No s'espera afegir funcionalitat nova gran.
