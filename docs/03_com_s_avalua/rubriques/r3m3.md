# R3M3. Primer cas d'ús complet

## RA avaluat

- **RA avaluat**: `RA8`
- **CA avaluats**: `RA8.d`, `RA8.e`, `RA8.g`
- **RA de context**: `RA5.e`, `RA5.f`, `RA5.g`, `RA6.c`, `RA6.d`, `RA6.f`

Este microrepte genera nota només per al RA avaluat. Els RA de context poden aparéixer en el producte, però no es ponderen en esta rúbrica.


## Visor del microrepte

| Camp | Valor |
|---|---|
| Repte | `R3` |
| Sessió | `R3S3` |
| Microrepte | `R3M3` |
| Pes dins del repte | `18%` |
| Autocorrecció | `r3-s03-primer-cas-us-complet` |

## Criteris i pesos

| Dimensió | Pes | Què es comprovarà |
|---|---:|---|
| Resolució funcional | `30%` | Flux heretat de `R2` amb ruta, controlador, BBDD, resposta i error mínim. |
| Verificació | `20%` | Cas positiu i error o validació controlada. |
| Traçabilitat | `10%` | Commits o registre del flux i la prova. |
| Documentació | `15%` | Quin flux ve de `R2`, com s'executa i quines dades usa. |
| Ús d'IA | `5%` | Registre si s'han acceptat esquelets o fragments. |
| Qualitat tècnica | `20%` | Responsabilitats recognoscibles segons el framework. |

## Evidències requerides

- Flux de `R2` identificat.
- Ruta i controlador.
- Model, servei o capa de dades.
- BBDD real.
- Vista, plantilla o resposta visible.
- Validació o error controlat.

## Fora d'abast

No es demana encara segon flux ni API.
