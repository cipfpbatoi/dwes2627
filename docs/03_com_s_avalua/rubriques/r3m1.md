# R3M1. Arrencada del nou projecte

## RA avaluat

- **RA avaluat**: `RA5`
- **CA avaluats**: `RA5.a`, `RA5.b`, `RA5.c`, `RA5.d`, `RA5.e`
- **RA de context**: `RA8.c`

Este microrepte genera nota només per al RA avaluat. Els RA de context poden aparéixer en el producte, però no es ponderen en esta rúbrica.


## Visor del microrepte

| Camp | Valor |
|---|---|
| Repte | `R3` |
| Sessió | `R3S1` |
| Microrepte | `R3M1` |
| Pes dins del repte | `12%` |
| Autocorrecció | `r3-s01-arrencada-nou-projecte` |

## Criteris i pesos

| Dimensió | Pes | Què es comprovarà |
|---|---:|---|
| Resolució funcional | `30%` | Projecte en framework arrancable, configuració mínima i ruta inicial funcional. |
| Verificació | `20%` | Arrencada i ruta comprovades seguint instruccions. |
| Traçabilitat | `10%` | Commits o registre d'inici, arrencada i tancament. |
| Documentació | `15%` | Comparativa, esquema abans/després, `.env.example` i dos casos d'ús. |
| Ús d'IA | `5%` | Registre i verificació si la IA ha intervingut. |
| Qualitat tècnica | `20%` | Ús real de convencions del framework i absència de secrets versionats. |

## Evidències requerides

- Projecte arrancable.
- Docker o entorn equivalent.
- `.env.example` o instruccions.
- Ruta inicial o `healthcheck`.
- Comparativa de stack.
- Esquema abans/després.
- Dos casos d'ús de `R3`.

## Fora d'abast

No es demana encara BBDD, migracions/fixtures/seeders, flux complet ni API.
