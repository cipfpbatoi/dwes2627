# R3M5. Qualitat i estabilització

## Visor del microrepte

| Camp | Valor |
|---|---|
| Repte | `R3` |
| Sessió | `R3S5` |
| Microrepte | `R3M5` |
| Pes dins del repte | `15%` |
| Autocorrecció | `r3-s05-qualitat-estabilitzacio` |

## Criteris i pesos

| Dimensió | Pes | Què es comprovarà |
|---|---:|---|
| Resolució funcional | `30%` | Dos fluxos funcionen amb BBDD i errors controlats. |
| Verificació | `20%` | Checklist o proves amb cas positiu, negatiu i migrations/seeders. |
| Traçabilitat | `10%` | Incidències, correccions i commits. |
| Documentació | `15%` | Proves, errors i estat dels fluxos. |
| Ús d'IA | `5%` | Registre si s'ha usat per depurar o proposar proves. |
| Qualitat tècnica | `20%` | Revisió de responsabilitats i estabilització real. |

## Evidències requerides

- Checklist o proves.
- Cas positiu i negatiu.
- Error controlat.
- Migrations i seeders comprovats.
- Incidències corregides.

## Fora d'abast

No s'espera afegir funcionalitat nova gran.
