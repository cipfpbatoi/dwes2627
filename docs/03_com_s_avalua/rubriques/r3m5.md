# R3M5. Autenticació, autorització i middleware

## RA avaluat

- **RA avaluat**: `RA5`
- **CA avaluats**: `RA5.f`, `RA5.g`, `RA5.h`
- **RA de context**: `RA6.f`, `RA8.g`

Este microrepte genera nota només per al RA avaluat. Els RA de context poden aparéixer en el producte, però no es ponderen en esta rúbrica.


## Visor del microrepte

| Camp | Valor |
|---|---|
| Repte | `R3` |
| Sessió | `R3S5` |
| Microrepte | `R3M5` |
| Pes dins del repte | `15%` |
| Autocorrecció | `r3-s05-autenticacio-autoritzacio-middleware` |

## Criteris i pesos

| Dimensió | Pes | Què es comprovarà |
|---|---:|---|
| Resolució funcional | `30%` | Login o autenticació funcional, acció protegida i autorització mínima. |
| Verificació | `20%` | Cas sense autenticar, cas autoritzat, cas denegat i logout o invalidació. |
| Traçabilitat | `10%` | Commits o registre de decisions de seguretat. |
| Documentació | `15%` | Usuari demo, ruta protegida, regla d'autorització i passos de prova. |
| Ús d'IA | `5%` | Registre si s'ha usat per proposar middleware, guard, voter o policy. |
| Qualitat tècnica | `20%` | La protecció està en servidor i usa convencions del framework. |

## Evidències requerides

- Login o mecanisme d'autenticació funcional.
- Logout o invalidació de sessió/token.
- Usuari demo reproduïble.
- Ruta o acció protegida.
- Middleware, guard, voter, policy o equivalent.
- Regla d'autorització mínima vinculada al domini.
- Cas autoritzat i cas denegat.
- Error o resposta controlada quan no hi ha permisos.

## Fora d'abast

No es demana un sistema complet de rols, OAuth, 2FA, JWT avançat ni API completa.
