# Rúbrica R5M4. Funcionalitat híbrida integrada

| Dimensió | Pes | Què es comprova |
|---|---:|---|
| Resolució funcional | `40%` | La integració funciona dins d'un flux real del producte. |
| Verificació | `20%` | La demo mostra flux complet, abans/després i error controlat. |
| Traçabilitat | `10%` | Commit rellevant comentat i registre d'incidències. |
| Documentació | `10%` | Instruccions suficients per reproduir la demo. |
| Ús d'IA | `5%` | Si hi ha IA, es registren decisions i validació real. |
| Qualitat tècnica | `15%` | Reutilitza connector i mapping sense duplicació excessiva. |

## Regles dures

- Si la integració no apareix en el producte, màxim recomanat `6`.
- Si no hi ha demo reproduïble, màxim recomanat `7`.
- Sense control d'error extern, màxim recomanat `8`.
