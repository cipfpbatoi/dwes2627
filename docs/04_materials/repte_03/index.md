# Materials del Repte 3

Estos materials són per treballar `R3` a classe sense quedar-se en teoria sobre `MVC`.

En `R3` has de reconstruir el projecte en framework i deixar una base real:

- Docker funcional
- BBDD creada per migracions o equivalent
- dades inicials amb fixtures/seeders o script equivalent
- `2` casos d'ús funcionals
- almenys `1` cas server-rendered amb vista o plantilla
- acció protegida amb autenticació i autorització del framework
- validació, errors i proves mínimes
- README curt d'arrencada i ús

## Materials

- [Fitxa general](fitxa_general.md): què és `R3`, què has de construir i què no es demana.
- [Microreptes](microreptes.md): enunciat operatiu de `MP1` a `MP7`.
- [Suport tècnic mínim](suport_tecnic.md): rutes, controladors, vistes, models, BBDD, migracions, fixtures/seeders o scripts, Docker i proves.
- [Checklists](checklists.md): comprovacions abans de donar per tancat cada tram.

## Lectura recomanada

1. Llig primer la [fitxa general](fitxa_general.md).
2. Treballa cada microrepte amb [microreptes](microreptes.md).
3. Consulta [suport tècnic](suport_tecnic.md) quan no sàpies on va cada peça.
4. Abans d'entregar, passa els [checklists](checklists.md).
