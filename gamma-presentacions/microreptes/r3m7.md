- Codi: `R3M7`; sessió: `R3S7`.
- Diferència entre README real i relat embellit.
- Què ha d'incloure el tancament:
  - arrencada;
  - BBDD, migracions o equivalent i càrrega inicial reproduïble;
  - dos fluxos;
  - acció protegida i cas denegat;
  - decisions tècniques;
  - demo;
  - backlog;
  - possible recurs o endpoint per a `R4`.
- Errors habituals: README genèric, instruccions que no funcionen, ocultar parts no migrades, no documentar BBDD.
