# Brief Gamma. R5M3. Transformació, mapping i valor afegit

## RA avaluat

- **RA avaluat**: `RA9`
- **CA avaluats**: `RA9.d`, `RA9.e`
- **RA de context**: `RA6.c`, `RA6.d`

Este microrepte genera nota només per al RA avaluat. Els RA de context poden aparéixer en el producte, però no es ponderen en esta rúbrica.


## Objectiu

Presentar el microrepte `R5M3`: transformar la informació externa perquè encaixe amb el domini del projecte i aporte valor funcional.

## Punts clau

- Mostrar el JSON extern tal qual no és integrar.
- El mapping traduïx dades externes a conceptes del producte.
- Cal decidir què es conserva, què es descarta i per què.
- La comparativa abans/després fa visible el valor.
- La persistència només s'usa si té sentit.

## Evidències

- Taula de mapping.
- Codi de transformació.
- Filtrat, normalització o enriquiment.
- Comparativa font externa / resultat final.
- Justificació de valor.

## Pregunta de tancament

Què canvia entre la resposta externa original i el resultat que usa el teu producte?
