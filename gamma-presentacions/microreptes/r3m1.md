# Brief Gamma. R3M1. Arrencada del nou projecte

## Objectiu

Presentar el microrepte `R3M1`: crear una base en framework arrancable, amb Docker o equivalent, `.env.example`, ruta inicial, comparativa de stack i abast dels dos fluxos de `R3`.

## Contingut

- Codi: `R3M1`; sessió: `R3S1`.
- Per què `R3` no és migrar tota l'aplicació.
- Què vol dir projecte arrancable en framework.
- Què ha de construir:
  - stack triat;
  - Docker o entorn equivalent;
  - `.env.example`;
  - ruta inicial o `healthcheck`;
  - comparativa curta;
  - esquema abans/després;
  - issue de `R3` amb dos fluxos.
- Evidències i checklist.
- Errors habituals: framework que no arranca, comparativa genèrica, secrets en `.env.example`, no declarar fluxos.

## Límit

No demanar encara BBDD, migrations, seeders ni flux complet.
