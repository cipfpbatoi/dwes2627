# Brief Gamma. R2M6. Proves, depuració i checkpoint tècnic

## Objectiu

Crear o actualitzar una presentació de microrepte per a `R2M6`, centrada en provar el flux complet de `R2`, registrar resultats reals, depurar incidències i deixar una demo reproduïble.

## Referències existents

- `docs/02_reptes/repte_02.md`
- `dwes-restructuracio-modul/docs/01_programacio_modul/programacio_aula_r2s6_proves_depuracio_checkpoint.md`
- `dwes-microreptes-autocorreccio/microreptes/r2-s06-proves-depuracio-checkpoint/`

## Contingut que ha d'incloure

- Codi del microrepte: `R2M6`.
- Sessió associada: `R2S6`.
- Per què provar no és només fer una demo del camí feliç.
- Bateria mínima:
  - cas vàlid;
  - cas invàlid;
  - reintent;
  - regla amb dos resultats;
  - estat recuperat i invalidat;
  - accés no autenticat denegat;
  - accés autenticat permés;
  - logout o invalidació.
- Com registrar entrada, passos, resultat esperat, resultat real i incidència.
- Errors habituals: proves no executades, README desactualitzat, corregir sense reproduir, no provar denegacions.
- Checklist final.

## Límit

No demanar proves automatitzades com a mínim ni funcionalitats noves.
