# Briefs Gamma. Microtallers

Este directori conté els briefs per generar presentacions curtes de microtallers amb Gamma.

Els microtallers són intervencions de `45-75` minuts dins de sessions de repte. No són microreptes nous ni tenen nota separada: ajuden a desbloquejar decisions tècniques i tanquen amb una evidència menuda al repositori.
